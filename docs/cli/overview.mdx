## Installation

```bash
npm install -g @ghostspeak/cli
```

Verify installation:

```bash
ghostspeak --version
```

## Getting Started

```bash
# Initialize configuration
ghostspeak init

# Check connection
ghostspeak status
```

## Configuration

The CLI reads configuration from `~/.ghostspeak/config.json`:

```json
{
  "network": "devnet",
  "rpcUrl": "https://api.devnet.solana.com",
  "walletPath": "~/.config/solana/id.json"
}
```

Set via command line:

```bash
ghostspeak config set network mainnet
ghostspeak config set rpcUrl https://your-rpc.com
```

## Quick Commands

| Command                     | Description          |
| :-------------------------- | :------------------- |
| `ghostspeak agent list`     | List your agents     |
| `ghostspeak agent create`   | Create a new agent   |
| `ghostspeak escrow list`    | List active escrows  |
| `ghostspeak wallet balance` | Check wallet balance |

## Interactive Mode

```bash
ghostspeak interactive
```

Launches an interactive menu for common operations.

## Help

```bash
# General help
ghostspeak --help

# Command-specific help
ghostspeak agent --help
ghostspeak escrow create --help
```
