## How Revenue Share Works

GhostSpeak protocol fees are distributed to stakers:

```
┌─────────────────────────────────────────────────────────────┐
│  Protocol Revenue Sources                                    │
├─────────────────────────────────────────────────────────────┤
│  1. B2C Pay-Per-Check (1 USDC/verification)                 │
│     └─> 10% → Staker rewards pool                           │
│                                                              │
│  2. B2B API Overage Fees (0.01 USDC/request over quota)     │
│     └─> 100% → Staker rewards pool                          │
│                                                              │
│  3. GHOST Burn Fees (75 GHOST burned per verification)      │
│     └─> Deflationary (no distribution)                      │
└─────────────────────────────────────────────────────────────┘

                              │
                              ▼
                    ┌─────────────────────┐
                    │  Rewards Pool (USDC) │
                    └─────────────────────┘
                              │
                              ▼
          ┌───────────────────────────────────────┐
          │  Distributed to Stakers (Proportional) │
          │  Based on: Stake × Multiplier          │
          └───────────────────────────────────────┘
                              │
          ┌───────────────────┴───────────────────┐
          ▼                                       ▼
   Verified Staker                          Whale Staker
   (1.5x multiplier)                        (3.0x multiplier)
   Stake: 5K GHOST                          Stake: 500K GHOST
   Effective: 7.5K                          Effective: 1.5M
```

### Distribution Formula

```typescript
// Calculate your USDC earnings
function calculateRewards(
  yourStake: number,        // GHOST tokens you staked
  yourMultiplier: number,   // 1.5x, 2.0x, or 3.0x
  totalEffectiveStake: number, // Sum of all (stake × multiplier)
  rewardsPool: number       // Total USDC in rewards pool
): number {
  const yourEffectiveStake = yourStake × yourMultiplier
  const yourShare = yourEffectiveStake / totalEffectiveStake
  return rewardsPool × yourShare
}

// Example
const rewards = calculateRewards(
  50_000,      // Your stake
  2.0,         // Pro Staker multiplier
  10_000_000,  // Total effective stake
  10_000       // $10K rewards pool
)
console.log('Monthly earnings:', rewards) // $100 USDC
```

## Claiming Rewards

Rewards accrue **daily** and can be claimed **anytime**:

<CodeGroup>

```typescript TypeScript SDK
// Check pending rewards
const rewards = await client.staking.getRewards()

console.log('Pending USDC:', rewards.pending)
console.log('Estimated monthly:', rewards.estimatedMonthly)
console.log('Estimated APY:', `${rewards.estimatedAPY}%`)

// Claim rewards
const result = await client.staking.claimRewards()

console.log('✅ Claimed:', result.usdcAmount, 'USDC')
console.log('Transaction:', result.signature)
```

```bash CLI
# Check rewards
```

<Tip>
**Gas-Free Claims**: Reward claims cost ~0.0005 SOL in transaction fees. Claim regularly to avoid accumulating too much pending balance.

## Unstaking

Unstake **anytime** with no lockup period:

```typescript
// Unstake all GHOST
const result = await client.staking.unstake({
  amount: 'all' // Or specify amount: 1_000_000_000n for 1K GHOST
})

console.log('✅ Unstaked:', result.amountGHOST, 'GHOST')
console.log('Transaction:', result.signature)

// Consequences:
// - Lose unlimited verification access
// - Stop earning revenue share
// - Keep all previously claimed USDC rewards
// - GHOST tokens returned to wallet immediately
```

<Warning>
**Unstaking Consequences**:
- ❌ Lose unlimited verification access (revert to pay-per-check)
- ❌ Stop earning revenue share
- ✅ Keep all claimed rewards
- ✅ GHOST returned to wallet immediately (no cooldown)
</Warning>

## Revenue Share Multipliers

Higher stakes earn higher multipliers:

```typescript
// Example: $10K rewards pool, 10M effective stake total

// Scenario 1: Verified Staker (5K GHOST, 1.5x multiplier)
Effective Stake = 5,000 × 1.5 = 7,500
Share = 7,500 / 10,000,000 = 0.075%
Earnings = $10,000 × 0.00075 = $7.50 USDC

// Scenario 2: Pro Staker (50K GHOST, 2.0x multiplier)
Effective Stake = 50,000 × 2.0 = 100,000
Share = 100,000 / 10,000,000 = 1.0%
Earnings = $10,000 × 0.01 = $100 USDC

// Scenario 3: Whale Staker (500K GHOST, 3.0x multiplier)
Effective Stake = 500,000 × 3.0 = 1,500,000
Share = 1,500,000 / 10,000,000 = 15.0%
Earnings = $10,000 × 0.15 = $1,500 USDC
```

**Key Insight**: Whale Stakers earn **200x more** than Verified Stakers for **100x more** stake (due to 3x vs 1.5x multiplier).
