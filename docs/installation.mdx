## System Requirements

- **Node.js** 18.0 or higher
- **Bun** 1.0+ (recommended) or npm/pnpm
- **Solana CLI** 2.1.0+ (for deployment)
- **Rust** 1.75+ (for smart contract development)

## SDK Installation

<CodeGroup>

```bash npm
npm install @ghostspeak/sdk
```

```bash bun
bun add @ghostspeak/sdk
```

```bash pnpm
pnpm add @ghostspeak/sdk
```

</CodeGroup>

### TypeScript Configuration

Ensure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "strict": true
  }
}
```

## CLI Installation

Install the CLI globally for easy access:

<CodeGroup>

```bash npm
npm install -g @ghostspeak/cli
```

```bash bun
bun add -g @ghostspeak/cli
```

</CodeGroup>

Verify the installation:

```bash
ghostspeak --version
```

## Solana Dependencies

For interacting with Solana, install the web3 libraries:

```bash
bun add @solana/kit @solana/signers
```

## Development Setup

Clone the repository for local development:

```bash
git clone https://github.com/ghostspeak/ghostspeak.git
cd ghostspeak
bun install
```

### Project Structure

```
ghostspeak/
├── packages/
│   ├── sdk-typescript/    # TypeScript SDK
│   ├── cli/               # Command-line interface
│   └── web/               # Web application
├── programs/              # Rust smart contracts
├── docs/                  # Documentation (you are here!)
└── scripts/               # Utility scripts
```

## Environment Setup

Create a `.env` file with your configuration:

```bash
# Solana RPC endpoint
SOLANA_RPC_URL=https://api.devnet.solana.com

# Wallet path (optional, can also use environment)
WALLET_PATH=~/.config/solana/id.json

# Program ID (use default for devnet)
GHOSTSPEAK_PROGRAM_ID=GpvFxus2eecFKcqa2bhxXeRjpstPeCEJNX216TQCcNC9
```

## Verify Installation

Test that everything is working:

```typescript
import { GhostSpeakClient } from '@ghostspeak/sdk';

const client = new GhostSpeakClient({
  rpcUrl: 'https://api.devnet.solana.com',
  commitment: 'confirmed',
});

// Fetch protocol info
const info = await client.getProtocolInfo();
console.log('Protocol connected:', info);
```
