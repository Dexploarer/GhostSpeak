## Overview

The Governance module enables programmatic interaction with the GhostSpeak DAO. Use this to create proposals, cast votes, and execute approved changes.

## Browser-Safe Import

```typescript
import {
  GovernanceModule,
  // Types defined in the module
} from '@ghostspeak/sdk/browser';
```

## Using the Governance Module

```typescript
import { GovernanceModule } from '@ghostspeak/sdk/browser';

const governanceModule = new GovernanceModule({
  programId: GHOSTSPEAK_PROGRAM_ID,
  rpcEndpoint: 'https://api.devnet.solana.com',
});
```

## Proposals

### Create Proposal

```typescript
const tx = await governanceModule.createProposal({
  title: "Increase Agent Name Length",
  description: "Update the protocol to allow 128-char agent names",
  category: "protocol-parameter",
  executionData: {
    instructionIndex: 2,
    data: Buffer.from([...]), // Serialized instruction data
  },
  signer: wallet,
});

console.log('Proposal created:', tx);
```

### Get All Proposals

```typescript
const proposals = await governanceModule.getAllProposals();

proposals.forEach(({ address, data }) => {
  console.log('ID:', address);
  console.log('Status:', data.status); // draft, active, succeeded, defeated, executed
  console.log('Votes For:', data.forVotes);
  console.log('Votes Against:', data.againstVotes);
});
```

### Get Proposal by ID

```typescript
const proposal = await governanceModule.getProposal(proposalAddress);

if (proposal.status === 'active') {
  console.log('Voting ends:', new Date(Number(proposal.votingEndsAt) * 1000));
}
```

## Voting

### Cast Vote

```typescript
await governanceModule.vote({
  proposal: proposalAddress,
  vote: 'yes', // 'yes' | 'no' | 'abstain'
  signer: wallet,
});

console.log('Vote cast');
```

## Execution

### Execute Proposal

Once a proposal has passed and the timelock has expired, it can be executed.

```typescript
await governanceModule.execute({
  proposal: proposalAddress,
  signer: wallet,
});

console.log('Proposal executed');
```

## PDA Derivation

```typescript
const proposalAddress = await governanceModule.deriveProposalPda(
  proposerAddress,
  'proposal-seed-123'
);
```

## Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| Proposal Deposit | 10,000 $GHOST | Required to create proposal |
| Voting Period | 5 days | Duration of active voting |
| Timelock | 2 days | Delay before execution |
| Quorum | 10% | Minimum % of supply required |

<Note>
  Governance parameters themselves can be changed via a governance proposal.
</Note>
