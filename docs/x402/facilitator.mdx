## GhostSpeak x402 Facilitator

GhostSpeak operates a **production-ready x402 facilitator** at `www.ghostspeak.io`. Unlike other facilitators that only handle payment verification, GhostSpeak provides a complete **trust layer** with escrow, reputation, and dispute resolution.

## Facilitator Endpoints

All endpoints are publicly accessible at `https://www.ghostspeak.io/api/x402`:

| Endpoint              | Method | Description                            |
| --------------------- | ------ | -------------------------------------- |
| `/api/x402`           | GET    | Facilitator info and capabilities      |
| `/api/x402/verify`    | POST   | Verify a payment on Solana             |
| `/api/x402/settle`    | POST   | Settle a payment and record reputation |
| `/api/x402/resources` | GET    | Discover available x402 resources      |
| `/api/x402/health`    | GET    | Health check and network status        |

## Quick Start

### 1. Configure Your Server

Set the facilitator URL in your server configuration:

```bash
FACILITATOR_URL=https://www.ghostspeak.io/api/x402
```

### 2. Verify Payments

```typescript
// Verify x402 payment
const response = await fetch('https://www.ghostspeak.io/api/x402/verify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    paymentHeader: 'signature:payload',
    requirement: {
      scheme: 'exact',
      network: 'solana',
      maxAmountRequired: '1000000', // 1 USDC
      resource: 'https://your-api.com/endpoint',
      payTo: 'YOUR_WALLET_ADDRESS',
      asset: 'EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v',
    },
  }),
});

const result = await response.json();
// { valid: true, payer: '...', amount: '1000000', transaction: '...' }
```

### 3. Settle and Record Reputation

```typescript
// Settle payment (records reputation on-chain)
const response = await fetch('https://www.ghostspeak.io/api/x402/settle', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    paymentHeader: 'signature:payload',
    requirement: {
      /* same as verify */
    },
    agentId: 'my-agent-id', // Optional: records reputation
    responseTimeMs: 150, // Optional: tracks performance
  }),
});
```
